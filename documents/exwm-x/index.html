1 Exwm-X

1.1 What is Exwm-X

Exwm-X is an extension of exwm (emacs x window manager), which can make exwm easier for Mouse-Control-People to use.

1.2 Feature

  1. Shortcuts, toolbar and other window operate buttons in mode-line.
  2. Move or resize a floating-window without press WIN key.
  3. Jump-or-exec, which will switch to an exist app instead of launch it again.

1.5 Configure

(require 'exwm-x)
(require 'exwm-x-example) ;; Adjust this line.

Note: exwm-x-example is Exwm-X buildin example, usr can use it to test Exwm-X's features. If it doesn't suit for your need, just copy and paste its useful pieces to your own exwm config :-)
